# sistema-controle-financeiro
Sistema de Controle Financeiro para organizar gastos e recebimentos por usuário e categoria. Banco Oracle com tabelas USUARIO, CATEGORIA, GASTO e RECEBIMENTO, integridade via PK/FK/CHECK e IDs automáticos com SEQUENCES e TRIGGERS. Scripts separados por pastas e arquivo 00_run_all.sql para criar tudo na ordem e testar.
